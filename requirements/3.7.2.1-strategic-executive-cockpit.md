# 3.7.2.1 Strategic Executive Cockpit — Frontend Requirements

**Source:** Section 3.7.2 — Strategic Decision-Making Layer  
**Audience:** Group / executives  
**Route:** `/bi/executive`

---

## Layout

Full-width dashboard grid — `max-w-[1920px]`, dark theme default for large displays.

### Global filter bar

Time: Year / Quarter / Month. Company, plant, product category, customer category, region.

---

## KPI row (8 cards)

| KPI | Format |
|-----|--------|
| Group sales revenue | Currency, tabular nums |
| Gross profit margin | % |
| Net profit | Currency |
| Order delivery rate | % |
| Production capacity utilization | % |
| Inventory turnover rate | × |
| ROI | % |
| OLE | % |
| KPI performance pass rate | % |

Each card: value, vs prior period %, sparkline.

---

## Charts

| Chart | Type | Interaction |
|-------|------|-------------|
| Financial trends | Combo bar + line | Drill by quarter |
| Customer profit contribution | Pareto | Click bar → filter region chart |
| Enterprise health | Radar | 6 axes mock |
| Market distribution | Map (simplified choropleth or region blocks) | Hover tooltip |

**Linkage:** pie/map selection filters all other charts on page (`BiLinkageProvider`).

---

## Drill-down path

Group → Company → Plant → Production line → Work order (breadcrumb navigation).

---

## Refresh

Micro-batch: configurable 5–30 min (settings link). Countdown in header.
