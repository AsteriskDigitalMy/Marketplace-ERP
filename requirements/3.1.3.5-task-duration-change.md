# 3.1.3.5 Project Task Duration Change Application — Frontend Requirements

**Source:** F-PROJ-005 — Section 3.1.3 Project Management  
**Route:** `/pms/tasks/:id/duration-change`

---

## Functional Summary

Executors request planned start/end adjustments; leader reviews. Approved changes update task duration with audit record.

**Prerequisites:** Task not completed.

---

## Application Form (executor)

| Field | Required |
|-------|----------|
| Adjusted planned start | Yes |
| Adjusted planned end | Yes |
| Reason | Yes — min length |

**Validation:** new dates not earlier than system current date (mock).

Submit → leader review queue.

---

## Leader Review (`/pms/projects/:id/duration-requests`)

Approve / Reject with opinion. Approve → task dates updated + change record.

---
