# 3.1.1.7 Fast Binding of User Roles — Frontend Requirements

**Source:** F-BASE-007 — Section 3.1.1 System Basic Management  
**Route:** `/pms/admin/accounts/:id/roles`

---

## Functional Summary

Individually or batch-bind business roles to employee accounts. Multi-role per user supported; permissions refresh instantly.

**Prerequisites:** User management permission.

---

## Single User Role Config

On account detail → **Roles** tab:

- Checkbox list of available roles ([3.1.1.6](./3.1.1.6-role-permission-configuration.md))
- Effective permission summary preview (read-only tree)
- Save → toast "Permissions refreshed" + change log

---

## Batch Binding (`/pms/admin/accounts` multi-select)

1. Select multiple accounts
2. **Batch assign roles** → dialog with role checkboxes (add/replace mode toggle)
3. Confirm → progress → summary
