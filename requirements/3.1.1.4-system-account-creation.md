# 3.1.1.4 System Account Creation — Frontend Requirements

**Source:** F-BASE-004 — Section 3.1.1 System Basic Management  
**Routes:** `/pms/admin/accounts`, `/pms/admin/accounts/import`

---

## Functional Summary

Create accounts manually (single) or bulk via Excel import with line-by-line validation.

**Prerequisites:** System user management permission.

---

## Single Create (`/pms/admin/accounts/new`)

| Field | Required | Validation |
|-------|----------|------------|
| Employee name | Yes | |
| Employee ID | Yes | Globally unique |
| Department | Yes | Org tree picker |
| Position/title | Yes | |
| Initial password | Yes | Strength rules (mock) |
| Bound role(s) | Yes | Valid preset roles |

Submit → mock login account generated + password notification toast.

---

## Bulk Import (`/pms/admin/accounts/import`)

1. Download official template (mock `.xlsx` link)
2. Upload file → validation progress
3. Results table: row #, status (success/fail), error message

Summary chips: N success, N failed. Failed rows downloadable (mock).

---
