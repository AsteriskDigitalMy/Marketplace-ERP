# 3.6.7 Inventory and Stock Valuation Integration — Frontend Requirements

**Source:** Section 3.6.7

---

## Process coverage (`/sap/inventory`)

| Transaction type | Ops source | SAP MM result |
|------------------|------------|---------------|
| All inventory movements | WMS docs | Material doc + valuation |
| Transfer | WMS transfer | Transfer voucher |
| Physical adjustment | Stock-taking | Gain/loss voucher |
| Status change | Quality/freeze | Block/unblock value |

Unified document list with transaction type filter.

---

### Transaction detail drawer

Payload: doc #, material, batch, qty, amount, plant, bin, type.

SAP material document # and posting date.

---

### Month-end inventory reconciliation (`/sap/inventory/reconciliation`)

Compare operational inventory ledger vs SAP stock subsidiary ledger.

| Warehouse | Material | Batch | Ops qty | SAP qty | Qty var | Ops value | SAP value | Value var |

Drill-down by warehouse, material, batch.

**Alert** on variance → finance + warehouse managers.

---

### Daily sync summary

Card: transactions synced today, failed, total value moved.
