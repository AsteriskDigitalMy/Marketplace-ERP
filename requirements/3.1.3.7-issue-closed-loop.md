# 3.1.3.7 Issue Assignment and Closed-Loop Handling — Frontend Requirements

**Source:** F-PROJ-007 — Section 3.1.3 Project Management  
**Route:** `/pms/projects/:id/issues`

---

## Functional Summary

Project leaders assign handlers, set deadlines, track rectification, and close verified issues. Full disposal workflow log retained.

**Prerequisites:** Logged-in user is project leader.

---

## Issue Aggregator

Table: Issue ID, Task, Description, Resource type, Status, Handler, Deadline, Actions.

Bulk assign: select issues → handler + deadline + resolution measures template.

---

## Handler Workspace

Assigned user fills disposal results; uploads evidence (mock).

---

## Leader Verification

| Decision | Effect |
|----------|--------|
| Close | Status `Closed`; log complete |
| Return | Back to handler with comments |

Timeline panel: reported → assigned → rectified → closed.
