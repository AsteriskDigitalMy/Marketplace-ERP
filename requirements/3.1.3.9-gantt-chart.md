# 3.1.3.9 Automatic Project Gantt Chart Generation — Frontend Requirements

**Source:** F-PROJ-009 — Section 3.1.3 Project Management  
**Route:** `/pms/projects/:id/gantt`

---

## Functional Summary

Auto-generated Gantt from planned and actual task durations. Timeline zoom, filter by executor, click drill-down, export HD image (mock).

**Prerequisites:** Project has sub-tasks ([3.1.3.3](./3.1.3.3-sub-task-creation.md)).

---

## Gantt View

Toggle from project detail → **Gantt** tab.

### Controls

- Timeline scale: day / week / month
- Filter by executor (multi-select)
- Legend: planned bar (blue), actual bar (green), overdue (red)

### Interactions

- Click task bar → task detail drawer (progress, owner, dates)
- Hover tooltip: name, %, dates
- **Export image** → mock PNG download

Responsive: horizontal scroll on narrow viewports.

---

interface GanttTaskDto {
