# 3.1.1.3 Organizational Structure Version Rollback and Recovery — Frontend Requirements

**Source:** F-BASE-003 — Section 3.1.1 System Basic Management  
**Route:** `/pms/admin/org/:id/history`

---

## Functional Summary

View historical org structure changes; preview any version; one-click rollback with new audit log.

**Prerequisites:** Architecture management or system audit permission.

---

## Version History Timeline

On org detail → **Version History** tab:

- Vertical timeline: version #, timestamp, actor, change summary
- Click version → side-by-side or full preview of tree at that point in time

### Preview panel

Read-only tree snapshot; highlight diff vs current (added/removed/renamed nodes — mock).

### Rollback

**Restore this version** → confirmation modal → progress → success toast + new log entry.
