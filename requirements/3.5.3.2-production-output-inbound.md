### Notice detail (`/wms/inbound/production/:id`)

- Work order metadata (mock from MES)
- Product barcode / RFID tag ID
- Quality inspection result panel — **OQC must be Passed** for put-away unlock
- Put-away task list with recommended finished-goods locations
- Generated production inbound document preview

---

## PDA Flow (`/wms/pda/inbound/production`)

### Step 1: Select task from MES notices list

### Step 2: Scan product barcode or RFID

Validation panel:

| Check | Result |
|-------|--------|
| Product identity match | ✓ / ✗ |
| Quantity match | ✓ / ✗ |
| OQC status | Must show Passed |

**Block put-away** if OQC ≠ Passed — destructive alert with message.

### Step 3: Put-away

- Recommended finished-goods location
- Scan location barcode
- Confirm qty

### Step 4: Complete

- Production inbound doc #
- MES work order inbound status → `Received` (mock badge)
- SCM + SAP sync status
