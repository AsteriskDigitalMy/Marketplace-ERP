# 3.5.8 Warehousing Statistics & Report Management — Frontend Requirements

**Source:** Section 3.5.8  
**Purpose:** Standard reports, custom report builder, multi-terminal viewports, export and subscription.

---

## Screens

### 1. Reports hub (`/wms/reports`)

**Two sections:**

#### Standard reports (pre-configured templates)

| Report | Route | Description |
|--------|-------|-------------|
| In/out daily summary | `/wms/reports/in-out-daily` | Daily ins and outs |
| In/out weekly | `.../in-out-weekly` | Weekly rollup |
| In/out monthly | `.../in-out-monthly` | Monthly rollup |
| Master stock ledger | `.../stock-ledger` | Full inventory ledger |
| Inventory aging | `.../aging` | Age buckets |
| Stagnant materials | `.../stagnant` | Idle stock |
| Material turnover | `.../turnover` | Turnover rates |
| Stock-taking variance | `.../count-variance` | From counting module |
| Worker efficiency | `.../worker-efficiency` | Operational logs by worker |

Each card: name, description, last run, `Run report` button.

---

### 2. Standard report viewer (shared layout)

**Filter bar** (varies by report):

- Date range
- Warehouse / zone
- Material category
- ABC class

**Output:**

- Summary KPI row
- Data table (paginated, tabular nums)
- Chart where applicable (shadcn Chart — bar, line, pie)

**Actions:** Export Excel, Export PDF, Print.

---

### 3. Custom report builder (`/wms/reports/custom`)

**Drag-and-drop builder** (MVP: configured sections form).

#### Dimensions (multi-select)

Warehouse, zone, location, material, category, batch, operator, transaction type, date.

#### Parameters

| Parameter | UI |
|-----------|-----|
| Date range | Date picker |
| Group by | Select |
| Sort by | Select |
| Filters | Dynamic filter rows |

#### Layout

- Choose columns to display (reorderable list)
- Chart type optional: none, bar, line, pie

**Save as template** — name + description → appears in "My templates" section.

---

### 4. My templates (`/wms/reports/templates`)

User-saved custom reports — run, edit, delete, duplicate.

---

### 5. Report subscriptions (`/wms/reports/subscriptions`)

| Column | |
|--------|--|
| Report | |
| Schedule | Daily / weekly / monthly + time |
| Recipients | Email list |
| Format | Excel / PDF |
| Last sent | |
| Status | Active / Paused |

**Create subscription form** — select report, schedule cron-like picker, recipient multi-email, format.

Mock "Send test now" button.

---

### 6. Multi-terminal viewports

| Viewport | Route | Layout |
|----------|-------|--------|
| PC admin | `/wms/reports/*` | Full filters + table + chart |
| PDA summary | `/wms/pda/reports` | Simplified daily in/out counts, open tasks |
| Dashboard | `/wms/reports/dashboard` | Large-screen KPI tiles + live charts (mock auto-refresh) |

#### Dashboard viewport (`/wms/reports/dashboard`)

- Full viewport layout (minimal chrome)
- KPI tiles: total stock value, today's in/out, alert count, occupancy %
- Live chart rotation every 30s (mock)
- Dark background option for warehouse TV display
