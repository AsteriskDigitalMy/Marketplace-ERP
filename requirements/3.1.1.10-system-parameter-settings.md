# 3.1.1.10 Custom System Parameter Settings — Frontend Requirements

**Source:** F-BASE-010 — Section 3.1.1 System Basic Management  
**Route:** `/pms/admin/parameters`

---

## Functional Summary

Configure global running rules and business control parameters. Immediate or scheduled effective time. Core parameter changes require double confirmation.

**Prerequisites:** Supreme system configuration permission.

---

## Parameter List

Table: Code, Name, Type (string/number/boolean/date), Current value, Effective at, Is core, Actions.

Filter: category, core only.

---

## Edit Dialog

| Field | Validation |
|-------|------------|
| Value | Must match parameter data type |
| Effective | Immediately / Scheduled datetime |

**Core parameters:** double-confirm modal before save.

Change log tab per parameter: old value, new value, actor, time (mock).

---
