### Edge Cases & Alternative Paths

- **Duplicate project name:** Inline error; submit blocked.
- **No enabled KPIs in library:** Empty state with link to [3.1.2.1](./3.1.2.1-kpi-indicator-addition.md).
- **Budget â‰¤ 0:** Validation error.
- **Team picker empty org users:** Link to [3.1.1.4](./3.1.1.4-system-account-creation.md).
- **Submit network failure:** Error toast with retry; preserve form state.
- **Returned project from [3.1.3.2](./3.1.3.2-project-initiation-approval.md):** Edit and resubmit flow from detail.
