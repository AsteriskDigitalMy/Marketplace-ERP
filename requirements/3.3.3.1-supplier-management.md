# 3.3.3.1 Supplier Management — Frontend Requirements

**Source:** Section 3.3.3.1

---

## Screens

### Supplier list (`/scm/suppliers`)

| Code | Name | Tier A/B/C/D | On-time % | IQC pass % | Status |

---

### Onboarding workflow

Steps: Info submission → Qualification review → Sample inspection → Small-batch trial → Qualified.

**Kanban or stepper** per supplier onboarding.

---

### Supplier profile

Master data, contacts, qualifications, supplied materials, pricing, payment terms, MOQ, lead times.

---

### Performance scorecard

Auto-scored metrics with tier badge. Tier C = restriction banner; Tier D = elimination confirm dialog.

---

### Supplier portal (mock `/portal/supplier`)

Suppliers view POs, ASN, IQC results, reconciliation (read-only mock).

---
