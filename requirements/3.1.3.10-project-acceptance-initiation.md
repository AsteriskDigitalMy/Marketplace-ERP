# 3.1.3.10 Project Overall Acceptance Application Initiation — Frontend Requirements

**Source:** F-PROJ-010 — Section 3.1.3 Project Management  
**Route:** `/pms/projects/:id/acceptance`

---

## Functional Summary

When all sub-tasks pass acceptance, leader submits overall project completion acceptance to auditor.

**Prerequisites:** All sub-tasks fully accepted.

---

## Pre-check Gate

Banner: "N of N tasks accepted" — submit disabled until 100%.

---

## Acceptance Application Form

| Field | Required |
|-------|----------|
| Overall completion summary | Yes |
| Target achievement status | Yes — met/partial/not met |
| Outcome attachments | Yes — mock upload |

Submit → status `Pending acceptance`; auditor queue +1 (mock).
