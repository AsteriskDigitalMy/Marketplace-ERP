## Functional Summary

Visual formula editor for industry-standard fixed formulas and custom logic. Reference indicators, system parameters, and math functions. One-click validation with simulated trial calculation.

**Prerequisites:** KPI indicator editing permission.

---

## Phase 1: Deep Requirement Analysis

### User Personas

| Persona | Role | Goals | Pain Points |
|---------|------|-------|-------------|
| **KPI Administrator** | Builds formulas | Use leather-industry presets or custom logic safely | Syntax errors; referencing disabled indicators |
| **Plant Analyst** | Validates formulas | See trial calculation output before publish | No visibility into referenced indicator health |
| **Developer (frontend)** | Implements editor | Clear embed contract with parent create/edit forms | Ambiguous save vs validate boundaries |

### Core Capabilities (FR-xxx)

| ID | Capability | Priority |
|----|------------|----------|
| FR-001 | Two modes: Fixed industry template picker vs Custom expression builder | Must |
| FR-002 | Toolbar inserts: enabled KPI indicators, system parameters, math functions | Must |
| FR-003 | Syntax-highlighted formula `Textarea` (mock highlighting) | Must |
| FR-004 | Searchable variable palette (`Command` + `ScrollArea`) | Must |
| FR-005 | "Validate formula" тЖТ result panel: pass/fail, simulated output, referenced indicator health | Must |
| FR-006 | Block parent save when referenced indicators disabled ([3.1.2.2](./3.1.2.2-kpi-indicator-lifecycle.md)) | Must |
| FR-007 | Apply formula to parent form on "Use formula" without full page save | Must |
| FR-008 | Embed mode in `Sheet` or `Dialog` from [3.1.2.1](./3.1.2.1-kpi-indicator-addition.md) | Should |

### Edge Cases & Alternative Paths

- **Template switch with unsaved custom edits:** `AlertDialog` confirm overwrite.
- **Insert disabled indicator:** Palette grays out disabled rows; insert blocked with toast.
- **Circular reference (AтЖТBтЖТA):** Validation error in result panel.
- **Empty formula validate:** Fail with "Formula required".
- **Validation API error:** Error state with retry `Button`.
- **Standalone route vs embed:** Standalone shows breadcrumb back to indicator detail; embed shows "Close" returning value to parent.

## Phase 2: Screen-by-Screen Functional Specifications

### Screen: Formula Editor тАФ `/pms/kpi/indicators/:id/formula` or embed

* **Objective:** Author, validate, and return a formula string to the parent indicator form.

* **UI Elements & Field-Level Specs:**
  - Layout: split pane тАФ left editor (60%), right palette + results (40%); stacks vertically on mobile.
  - **Mode toggle** тАФ `Tabs` or `ToggleGroup`: "Industry template" | "Custom logic".
  - **Industry template** тАФ `Select` of leather-industry presets (e.g. yield rate, on-time delivery); selecting populates formula `Textarea` (read-only until switch to custom).
  - **Formula editor** тАФ `Textarea` monospace, min-height 200px, mock syntax highlight via CSS classes; `Label` "Formula expression".
  - **Toolbar** тАФ `Button` group inserts at cursor: Indicators, Parameters, Functions (`DropdownMenu` per category).
  - **Variable palette** тАФ `Command` search; sections: KPI Indicators (enabled only), System Parameters ([3.1.1.10](./3.1.1.10-system-parameter-settings.md)), Functions (`SUM`, `AVG`, `MIN`, `MAX`, `IF`, `ROUND`, etc.); click inserts token at cursor.
  - **Validate formula** тАФ primary outline `Button`; shows `Loader2` while validating.
  - **Result panel** тАФ `Card`: status `Badge` pass/fail; `Alert` for errors list; simulated result number; `Table` of referenced indicators (Code, Name, Enabled status `Badge`).
  - Footer: `Button` "Cancel/Close"; `Button` "Use formula" (disabled until validation pass).

* **Functional Logic:**
  1. Load current formula from parent context or indicator snapshot.
  2. Validate calls mock `FormulaValidationResultDto`; renders errors and health table.
  3. "Use formula" emits string to parent ([3.1.2.1](./3.1.2.1-kpi-indicator-addition.md) / [3.1.2.2](./3.1.2.2-kpi-indicator-lifecycle.md)); closes `Sheet` if embedded.
  4. Referenced disabled indicators тЖТ `IsValid: false`; block "Use formula".

---

### Modal: Template Overwrite Confirm тАФ `AlertDialog`

* **Objective:** Warn when switching templates discards custom edits.

* **UI Elements & Field-Level Specs:** Standard confirm/cancel.

* **Functional Logic:** On confirm, replace `Textarea` content with template formula; reset validation state.

---

### Panel: Function Reference тАФ `Popover` or `Sheet`

* **Objective:** Quick reference for function signatures.

* **UI Elements & Field-Level Specs:** `ScrollArea` list of functions with signature and description; click inserts.

* **Functional Logic:** Non-blocking helper; does not require validation.
