# 3.1.2.3 KPI Formula Visual Editor — Frontend Requirements

**Source:** F-KPI-003 — Section 3.1.2 KPI Indicator Library Management  
**Route:** `/pms/kpi/indicators/:id/formula` (embedded in create/edit)

---

## Functional Summary

Visual formula editor for industry-standard fixed formulas and custom logic. Reference indicators, system parameters, and math functions. One-click validation with simulated trial calculation.

**Prerequisites:** KPI indicator editing permission.

---

## Editor Modes

| Mode | Description |
|------|-------------|
| Fixed industry formula | Preset template picker (leather industry KPIs) |
| Custom logic | Free-form expression builder |

### Toolbar inserts

- Existing KPI indicators (enabled only)
- System fixed parameters
- Math functions: `SUM`, `AVG`, `MIN`, `MAX`, `IF`, `ROUND`, etc.

### Editor UI

- Text area with syntax highlighting (mock)
- Variable palette (searchable)
- **Validate formula** → result panel: pass/fail, simulated output, referenced indicator health

**Validation rules:** syntax compliant; referenced indicators must be enabled.
