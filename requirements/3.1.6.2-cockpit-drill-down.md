# 3.1.6.2 Cockpit Data Hierarchical Drill-Down Interaction — Frontend Requirements

**Source:** F-DASH-002 — Section 3.1.6 KPI Visualization Monitoring Dashboard  
**Route:** `/pms/cockpit` (interaction overlay)  
**Prerequisites:** Role cockpit page loaded ([3.1.6.1](./3.1.6.1-role-cockpit-auto-loading.md))

---

## Functional Summary

Click dashboard charts, data cards, or status color blocks to drill down layer by layer into detailed business data. Macro summary → departmental breakdown → team/group/individual performance.

**Input:** Unique identification ID of clicked data element.

---

## Click Targets

| Widget type | Click behavior |
|-------------|----------------|
| Data card | Open detail sheet with breakdown table |
| Chart segment / bar / point | Open detail sheet filtered to segment |
| Status color block | Open detail sheet for items in that status band |
| Table row | Open detail sheet for that entity |

All clickable elements show `cursor-pointer` and focus ring; keyboard Enter activates drill-down.

---

## Drill-Down Sheet / Dialog

Use shadcn `Sheet` or `Dialog` (wide on desktop, full-screen on mobile).

### Level 1 — Summary breakdown

- Title: parent metric name + breadcrumb (`Cockpit > Dept Achievement Rate`)
- Breakdown table: department / category rows with values and status colors
- Each row is clickable → Level 2

### Level 2 — Team / group detail

- Breadcrumb updates: `… > Production Dept > Team A`
- Table: team or group rows with KPI values, achievement %, responsible lead

### Level 3 — Individual performance

- Breadcrumb: `… > Team A > Individuals`
- Table: employee name, employee ID, individual KPI scores, rank
- Employee name link → optional employee detail popover (read-only mock profile)

### Navigation

- Breadcrumb chips navigate back up levels without closing sheet
- Close button returns to cockpit; preserve cockpit scroll position

---

## Loading States

- Sheet opens immediately with skeleton table
- Each level fetch is independent; show spinner in sheet body
- Empty level → "No detail data for this selection"

- [ ] Breadcrumb navigation between levels

- [ ] Keyboard accessible (focus, Enter, Escape to close)
