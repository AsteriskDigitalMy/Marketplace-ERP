# 3.1.3.6 One-Click Reporting of Project Execution Issues — Frontend Requirements

**Source:** F-PROJ-006 — Section 3.1.3 Project Management  
**Route:** Embedded in `/pms/tasks/my` progress form

---

## Functional Summary

Report on-site difficulties and resource needs during progress updates. Task marked at risk; project leader notified in real time.

**Prerequisites:** Sub-task `In Progress`.

---

## Issue Report Section (on progress update)

| Field | Required |
|-------|----------|
| Issue description | Yes |
| Required resource type | Yes — picker (personnel/equipment/material/budget/other) |

Checkbox: **"Flag task at risk"** (default on when issue filled).

Submit with progress → issue record created; task risk badge; leader notification toast.

---

## Leader Risk Feed (mock banner on project detail)

List of open issues with task link → routes to [3.1.3.7](./3.1.3.7-issue-closed-loop.md).
