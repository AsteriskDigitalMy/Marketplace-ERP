# 3.5.5.4 Inventory Reservation and Freezing Management — Frontend Requirements

**Source:** Section 3.5.5.4  
**Purpose:** Reservations for orders/work orders; operational locks; quality freezes — prevent double allocation.

---

## Screens

### 1. Reservation & freeze hub (`/wms/reservations`)

**Tabs:** Reservations | Operational locks | Quality freezes | Audit log

---

---

### Tab: Quality freezes

Materials failed IQC, pending inspection, or quality risk.

| Material | Batch | Qty | Freeze reason | Frozen at | Source |
|----------|-------|-----|---------------|-----------|--------|

**Auto-freeze banner** on related inbound screens when IQC fails.

**Lift freeze** — requires QC role (mock) + approval comment.

Unfreeze → stock returns to `Available` (status animation mock).

---

### Tab: Audit log

Immutable log of all lock/freeze/reserve/release events.

| Timestamp | Action | Entity | Operator | Note |
|-----------|--------|--------|----------|------|

Filters: action type, operator, date range.

---

## Ledger status visualization

On inventory views, show composite status:

`Available` = not reserved, not frozen, not locked  
`Reserved` = qty earmarked  
`Frozen` = quality hold  
`Locked` = operational hold

Stacked bar or badge combo on location stock tables.

## Mock Data Shape

```ts
interface InventoryReservationDto {
  Id: string
  ReferenceNumber: string
  ReferenceType: 'sales_order' | 'work_order'
  MaterialCode: string
  ReservedQty: number
  LocationCode: string
  Batch: string | null
  Status: 'active' | 'released'
  SourceModule: 'SCM' | 'MES'
}

interface OperationalLockDto {
  Id: string
  ScopeType: 'location' | 'stock_line' | 'zone'
  TargetCode: string
  Reason: string
  LockedBy: string
  LockedAt: string
  BlocksInbound: boolean
  BlocksOutbound: boolean
}

interface QualityFreezeDto {
  Id: string
  MaterialCode: string
  Batch: string
  Quantity: number
  FreezeReason: string
  FrozenAt: string
  Source: 'iqc_fail' | 'pending_inspection' | 'quality_risk'
  Status: 'active' | 'lifted'
}
```
