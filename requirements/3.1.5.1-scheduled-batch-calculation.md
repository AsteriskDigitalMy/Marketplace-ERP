# 3.1.5.1 Scheduled Automatic Batch Calculation — Frontend Requirements

**Source:** F-CALC-001 — Section 3.1.5 KPI Calculation and Statistical Analysis  
**Route:** `/pms/kpi/calculation/jobs`

---

## Functional Summary

System automatically batch-calculates all corporate KPIs on preset cycles. Pulls approved data from collection module; flags anomalies and pushes admin alerts.

**Prerequisites:** Formulas configured; baseline data reviewed and stored.

---

## Calculation Jobs Dashboard

| Column | Notes |
|--------|-------|
| Cycle | Daily/weekly/monthly/quarterly/annual |
| Scheduled at | Next run time |
| Last run | Timestamp + duration |
| Status | Success / Partial / Failed |
| Indicators | Total / success / anomaly count |

### Job detail

- Per-indicator result: value, target, status color
- Anomaly rows: error message, link to indicator ([3.1.2.2](./3.1.2.2-kpi-indicator-lifecycle.md))
- Error log download (mock)

Admin alert toast when anomalies detected (mock).
