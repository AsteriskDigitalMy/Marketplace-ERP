## Alert rules (`/bi/alerts/rules`)

Per-indicator threshold config:

| Field | |
|-------|--|
| Dashboard / KPI | Picker |
| Upper limit | Optional number |
| Lower limit | Optional number |
| Notify | User/role multi-select |
| Channels | In-app, email, mobile (mock) |

Example: scrappage rate > 3% → alert.

---

## Alert inbox (`/bi/alerts`)

| Severity | KPI | Actual | Threshold | Dashboard link | Status |

Open → acknowledge → resolved.

---

## Dashboard alert UI

When threshold breached:

- Chart/card **destructive border** pulse (respect `prefers-reduced-motion` — solid border only)
- Badge count on `/bi/alerts` nav item
- Mobile push mock toast
