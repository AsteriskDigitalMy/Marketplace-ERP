### Pick list detail (`/wms/outbound/picking/:id`)

- Materials table with recommended location, batch, pick path sequence
- Allocation strategy used (FIFO/FEFO badge)
- Pick log (operator, scans, timestamps)
- MES sync status

---

## PDA Flow (`/wms/pda/outbound/picking`)

### Step 1: Task list (MES-sourced, real-time mock)

### Step 2: Picking path view

Ordered list of pick stops:

| Stop | Location | Material | Batch | Qty to pick |
|------|----------|----------|-------|-------------|

Map-style simplified route (optional MVP: numbered list).

### Step 3: Barcode picking (per stop)

1. Scan location barcode — validate match
2. Scan material barcode — validate match
3. Enter picked qty
4. **Over-pick validation** — if qty > required, block with alert unless supervisor override code (mock dialog)

### Step 4: Outbound review

Scan material barcode + pick list barcode → confirmation screen.

### Step 5: Complete

- Production outbound document #
- Ledger deduction summary
- MES work order picking status updated (mock)
- SCM + SAP sync badges
