# 3.7.4 Custom Reporting and Self-Service Analysis — Frontend Requirements

**Source:** Section 3.7.4  
**Routes:** `/bi/reports/designer`, `/bi/reports`, `/bi/reports/subscriptions`

---

## Report designer (`/bi/reports/designer`)

### Drag-and-drop canvas (MVP)

**Left palette:**

- Dimensions: date, plant, supplier, customer, material, workshop, etc.
- Measures: sales volume, cost, profit, qty, rate %

Drag to **Rows**, **Columns**, or **Chart** drop zone.

**Chart type picker:** bar, line, pie, table.

Live preview panel updates on drop (mock aggregation).

---

### Save & publish

Name, description, default filters, shared vs private.

Saved to **My reports** list.

---

## Drill-down and linkage

All designer charts and built-in dashboards:

- **Bidirectional linkage** — selecting segment filters peer charts on same page
- **Vertical drill** — breadcrumb: Group → Company → Plant → Line → WO/Material

`BiDrillBreadcrumb` component — click level to zoom out.

---

## Export

Per report/chart: Excel, PDF, PNG image (mock download).

Print stylesheet for PDF.
