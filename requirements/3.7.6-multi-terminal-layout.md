# 3.7.6 UI Layout and Multi-Terminal Adaptation — Frontend Requirements

**Source:** Section 3.7.6

---

## Theme

**Dark / Light toggle** — persisted in `localStorage`.

- Dark default for executive cockpit and kanban
- Light default for PC analyst workflows
- High contrast, industrial style — minimal decorative animation
- Use shadcn semantic tokens only (`bg-background`, `text-foreground`, etc.)

---

## Terminal layouts

### PC (`/bi/*` — default)

- Full sidebar + header
- `max-w-7xl` content for operational dashboards
- Report designer full width
- Deep analysis: filters always visible

### Large screen / TV (`/bi/kanban/*`, `/bi/executive?kiosk=1`)

- No sidebar/header chrome
- `100vw × 100vh`
- Auto carousel optional (`?carousel=1&interval=30`)
- Font scale +20% for viewing distance

### Mobile (`/bi/mobile`)

Responsive routes or breakpoints:

| Feature | Mobile |
|---------|--------|
| Core dashboards | Simplified single-column KPI + one chart |
| Alerts | Full list with push-style cards |
| Reports | View-only, no designer |
| Filters | `Sheet` bottom drawer |

Bottom nav: Home, Dashboards, Alerts, Profile.

---

## Permission enforcement (UI)

| Control | Behavior |
|---------|----------|
| Dashboard list | Filtered by role entitlements |
| Data scope | Auto-applied filters (region/plant) — read-only badge showing scope |
| Designer | Hidden for execution-tier roles |
| Export | Disabled if role lacks permission — tooltip explains |

Mock role switcher in dev toolbar for testing scopes.

---

## Responsive breakpoints

| Breakpoint | Layout |
|------------|--------|
| ≥ 1024px | Full dashboard grid |
| 768–1023px | 2-column grid, collapsible filters |
| < 768px | Mobile layout / `/bi/mobile` |
| ≥ 1920px | Executive multi-column expand |

Charts resize via responsive container; tables collapse to cards on mobile per UI rules.

---

## Acceptance Criteria

- [ ] Dark/light theme toggle with persistence
- [ ] PC, kiosk/TV, and mobile layout variants
- [ ] Role-based dashboard list and data scope filters
- [ ] Designer hidden for unauthorized roles
- [ ] Charts and tables adapt per breakpoint
- [ ] Four async states on all terminal variants
