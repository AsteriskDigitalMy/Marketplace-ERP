## PDA Execution (`/wms/pda/inbound/misc/:id`)

**Precondition:** approval status = Approved.

Per line:

1. Scan material barcode
2. Scan batch (if batch-managed)
3. Scan location barcode
4. Enter qty
5. Auto-capture operator + timestamp (display only)

**Complete** → generates inbound document type per category (mock PDF link).

---

## Business Rule UI

- Execute button disabled until approved — tooltip explains.
- All lines must be scanned — progress indicator `3/5 lines complete`.
- Sync panel: related module + SAP Financial (mock).

}
