# 3.3.3.4 Procurement Arrival and IQC Inspection — Frontend Requirements

**Source:** Section 3.3.3.4

---

## Flow UI

1. **Supplier ASN** — supplier portal logs dispatch → arrival notice (mock)
2. **WMS receive** — link to WMS purchase inbound (3.5.3.1)
3. **IQC request** — auto on warehouse verify
4. **IQC inspection** (`/scm/procurement/iqc`)

| Result | Next |
|--------|------|
| Pass | WMS put-away task |
| Fail | Return/isolation |
| Concession | Approval dialog |

5. **Supplier performance** — fail updates supplier scorecard

---

### IQC Pad/form

Inspection standards, NC qty, notes, photos (mock).

Disposition: return, rework, concession → purchase return bill.
